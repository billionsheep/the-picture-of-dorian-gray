# 游戏设计文档（GDD）——《格雷画像》绣湖式点击解谜（MVP）

## 1. 一句话概念（Logline）
玩家以“道林”的视角，在一系列封闭房间中点击、收集、组合与使用物品，逐步揭开“青春外表”背后的腐坏真相；每一次解谜都在推进道德分裂与自我审查的主题。

## 2. 核心体验（Rusty Lake-like）
- 低门槛：点一点就能推进
- 强叙事：每个物品/文本都暗示“分裂、审查、感官享乐”
- 小空间高密度：一个房间内完成一个闭环
- 轻度恐怖/怪诞：以“画像”作为恐怖源，逐步显形

## 3. MVP 目标（只做 1 关）
关卡：**巴兹尔的画室（Studio）**
完成一个闭环：
- 点击获得线索（信件/剪报/暗示）
- 线索转化为可用物品（密码）
- 用密码打开抽屉（获得挂钩/钥匙）
- 用挂钩触发“揭开画像”
- 出现结尾字幕（MVP 完成）

## 4. 交互系统（必须实现）
1) Scene：背景 + 热区（Hotspot）
2) Dialogue：文本弹窗（显示/关闭）
3) Inventory：物品栏（增删/选中）
4) Use Item：选中物品对热区使用（success/fail）
5) Flags：全局/关卡状态（如 drawer_open）

## 5. UI 规则（占位阶段）
- 背景：纯色或简单渐变；可选放置占位图路径但不要求存在
- 热区：透明矩形（调试时可显示边框）
- 对话框：屏幕底部半透明面板 + 文本
- 背包：底部横条，显示物品名（后期替换图标）

## 6. 文本风格
- 早期：自恋、审美化（“和谐”“美”）
- 后期：偏执、恐惧（“灵魂”“秘密”）
- 允许中英混排但优先中文（便于你写文档/展示）

## 7. 扩展关卡（非 MVP，可后续做）
- 剧院后台：绝笔信、调查报告（Prussic Acid）
- 图书室：黄色书、被涂改的信（审查机制）
- 学校实验室：化学销毁步骤（颜色/顺序）
- 鸦片馆：面具、身份切换

> 先把“系统”做成可扩展，后续内容就是配置与文本填充。
