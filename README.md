# the-picture-of-dorian-gray
项目名称： The Picture of Dorian Gray: A Digital Interactive Adaptation  类型： 2D 点击解谜游戏 (Point-and-Click Adventure)  风格参考： Rusty Lake (绣湖) / 极简主义 / 维多利亚哥特  技术栈 (Tech Stack)：  前端框架： React (使用 Vite 构建) —— Antigravity 最擅长的栈  状态管理： React Context API 或 Zustand —— 用于管理复杂的解谜状态  样式： Tailwind CSS —— 快速实现布局和UI  动画： Framer Motion —— 实现物品浮动、转场动画
